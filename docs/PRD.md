# 영화 리뷰 앱

## 1. 프로젝트 개요

### 1.1 프로젝트명

영화 리뷰 앱

### 1.2 프로젝트 목적

사용자가 영화 리뷰와 별점을 작성하고 관리할 수 있는 간단한 풀스택 웹 앱

### 1.3 개발 범위

- 영화 제목 등록 기능
- 영화 리뷰 작성 기능
- 별점 등록 기능
- 리뷰 수정 및 삭제 기능

## 2. 기술 스택

- 프런트엔드: HTML, CSS, 리액트, 자바스크립트
- 백엔드: Node.js, Express
- 데이터베이스: PostgreSQL

## 3. 기본 사항

- 프런트엔드와 백엔드를 따로 개발
- 기본적인 웹 기술만 사용
- 로그인 기능 제외
- 간단한 디자인으로 구현
